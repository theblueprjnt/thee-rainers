Primary Constraint → Prescribed Focus

Cue Reading

Slow sparring with cue-identification only. No counters. Just see the telegraphs.

Kinetic Chain

Single-punch kinetic drills: floor → hip → shoulder → fist. Pause at each node.

React. Stiffness

Single-leg loading, pogo hops, lateral bound-and-land. Build the elastic reload.

Range Fluency

Prescribed rounds at each range. No crossover. Long / mid / close rotations with purpose.

Def. Responsibility

After every combination: 3-count guard check. Ingrain the return before adding speed.

Transfer to Live

Pressure escalation: drilling → light contact → sparring. Same technique at every stage.

Identity & Presence

Adversity conditioning: start each round deliberately behind. Execute the system through it.
